# Queue with circular array implementation

This is a simple project to build a thread-safe FIFO queue based on a circular array, either in C or C++ language.

## Folders structure

- <i>build/</i> : build folder with Makefile
- <i>inc/</i> : headers/interfaces for the queue
- <i>src/</i> : C and C++ implementations
- <i>test/</i> : main file for tests
